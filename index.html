Glossary

This card collects terms and their definitions into a Description List Element (dl) for later reference.

tag name
The word inside an HTML tag that defines the element
tag
A tag name and opening and closing delimeters. An closing tag puts a forward slash in front of the tag name.
element
An opening tag, some content, and a closing tag. Some elements don't have closing tags and some elements are self closing.

Glossary 2

These terms relate to git and github

working directory
the directory you are in (usually that you are about to add to your repository)
staging area
An area where git stores a list of files before making a commit
repo (repository)
A colletion of all your commits
Remote
A cloud stored version of your repository, usually on a service like Github. You push to and pull from your remote.
